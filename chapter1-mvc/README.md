# Chapter1 - 關於MVC

MVC指是Ｍ(Model)、Ｖ(View)、Ｃ(Controller)，是軟體開發的一種架構模式，目的是將不同的行為切開來，對於程式碼撰寫與維護上可以比較有規則。而Rails的開發是遵循著MVC的架構的。

此系列將介紹在MVC中應該要知道的相關知識。
